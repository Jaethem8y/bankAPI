What is this API?

This API utilizes bank_db provided by Dr.Eric Manley to show the content without SQL queries. All the data are provided by JSON format

How to use this API

To get the data_dict table

address/data_dict

This returns the content of all the data_dict table in JSON

To get the fdic_fail table

address/fdicfail

This returns the content of all the fdic_fail table in JSON

address/fdicfail/{bank_id}

This returns the content of all the fdic_fail table that has same bank_id as bank_id in url in JSON

To get the other tables

address/table_{table name}

This returns the content of all the table name table in JSON

address/fdicfail/{bank_id}

This returns the content of all the table nametable that has same bank_id as bank_id in url in JSON

Example for other tables

address/CALL8786

This returns all the content from table_CALL8786 in JSON

address/CALL8786/28

This returns all the content from table_CALL8786 that has bank_id of 28 in JSON
